# Other Tactics

This directory contains tactic generation experiments beyond the main `limit_auto`
2×2 study, representing future work.

---

## `decide_list_theory/` — Second complete tactic

A complete tactic generation run using the **plan + LSP** strategy (Condition D).

**Target**: A tactic that decides list-theoretic propositions (membership, equality,
sublist, etc.) automatically.

**How it was run**: Interactively in **Claude Code** with live LSP tools enabled,
following the same methodology as `limit_auto` Condition D. The specification was first
generated using the pipeline's `analyze_request` stage, then a milestone plan was
written, and the model was asked to implement milestones one by one — feeding back the
previous milestone file and the plan at each step, with LSP access throughout.

**Files**:

| File | Description |
|------|-------------|
| `decide_list_theory.spec.md` | Mathematical specification (pipeline `analyze_request` output) |
| `decide_list_theory.plan.md` | 5-milestone development plan |
| `decide_list_theory.milestone1.lean` — `.milestone5.lean` | Incremental implementations |
| `lean_lsp_log_decide_list_theory.md` | Record of LSP tool calls during development |

**Status**: Complete. Provides a second data point for the plan + LSP strategy
beyond `limit_auto`.

## `specs/` — Draft specifications (future work)

Formalized specifications generated by the **pipeline's `analyze_request` stage**,
applied to various informal user descriptions. Not yet implemented as tactics.

| Spec file | Tactic concept |
|-----------|---------------|
| `nonzero.spec.md` | Automatically discharge nonzero side conditions |
| `poly_coeff.spec.md` | Extract polynomial coefficients |
| `poly_norm.spec.md` | Normalize polynomial expressions |
| `log_exp_order.spec.md` | Prove ordering facts about log/exp |
| `norm_num_bounds.spec.md` | Numeric bound verification extending `norm_num` |
| `presburger.spec.md` | Decision procedure for Presburger arithmetic |
| `real_poly_decide.spec.md` | Polynomial arithmetic over ℝ |
| `ring_char.spec.md` | Ring characteristic reasoning |
| `simp_leading_coeff.spec.md` | Simplify leading coefficient goals |
| `prove_exists_by_tendsto.spec.md` | Prove existence via limit arguments |
| `decide_monic.spec.md` | Decide monicity of polynomials |
| `generated_tactic.spec.md` | Generic tactic from pipeline batch run |

These specs were generated to test the pipeline's analysis stage and as candidates
for future implementation experiments.
